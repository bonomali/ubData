ubData

The purpose of this simple API is to allow for easier use of the air pollution data that exists for Ulaanbaatar, Mongolia. I was interested in working on a simple project and needed access to this data but it was not available in an easy to get at form. Therefore, I decided to go ahead and create something that would allow others access to the available data in an easy to use manner.

Data Source

Frequency

The instrument is scheduled to report 3 hour averages for a full day, running every other day.
