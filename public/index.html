ubData

Ulaanbaatar air pollution API

A simple API that gives easier access to the air pollution data that exists for Ulaanbaatar, Mongolia.

Fork me on GitHub!

Functional Description

Right now the API acts as a lightweight wrapper around the output of a single instrument. The instrument outputs something that looks like this to a public Dropbox folder

2013-05-05;00:00-03:00;0.024000;1;47.924364°N;106.918375°E;

and the API turns it into something that looks like this

{"startTime":1367726400000,"endTime":1367737200000,"pm25":24,"station":{"id":"1","latitude":47.924364,"longitude":106.918375}}

The API output format is in JSON, which is easier to work with programmatically and also takes care of saving data to a database.
