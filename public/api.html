ubData

Version 1

Resource Description Returns
GET /1/mostRecent Will get the most recent measurement available. Object containing time, PM2.5 value and location of measurement.
GET /1/sendTweet Will send out a tweet to the linked account. This is dependent on the key param matching a value stored on the server Returns success if a tweet was attempted to be sent, does not indicate if it was successfuly sent.

Address

All API calls should be made to ubdata.herokuapp.com.

Return format

All data is returned as JSON objects.
